#Installation

Note: Excel Export/Import - MySQL Data tool is depended on php web software

For using Excel Export/Import tool, first you have to confirm that, Excel Export/Import tool is already installed in your site. Also confirm that you have product available in your store. Then install Excel Export/Import tool.

Requirements
  • PHP version 7.1 or newer
  • PHP extension php_zip enabled
  • PHP extension php_xml enabled
  • PHP extension php_gd2 enabled
To install, Please follow the steps bellow:

Step-1:

  • Extract the archive file on your machine that you've downloaded from CodeCanyon.

The following screen will appear

  • Enter into the ExcelExportImport folder

The following screen will appear

Step-2:

  • Copy this folder ExportImportMySQLData_v2.3 and paste your localhost/web server directory

The following screen will appear

Step-3:

  • Database create as like(exportimport_db) in localhost/web server
  • Import database (demo_database/exportimport_db.sql)

The following screen will appear

Step-4:

  • Find the dbconnection.php page in ExportImportMySQLData_v2.3 folder
  • Then, change the database connection in (dbconnection.php) page

The following screen will appear

Step-5:

  • Then, browse this ExportImportMySQLData_v2.3 folder.
